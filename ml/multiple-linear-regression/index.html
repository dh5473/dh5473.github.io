мқҙм „ кёҖм—җм„ң кІҪмӮ¬н•ҳк°•лІ•мңјлЎң л©ҙм Ғ вҶ’ к°ҖкІ© мҳҲмёЎ лӘЁлҚёмқ„ л§Ңл“Өм—ҲлӢӨ. мһҳ лҸҷмһ‘н–Ҳм§Җл§Ң, н•ң к°Җм§Җ м°ңм°ңн•ң м җмқҙ мһҲм—ҲлӢӨ вҖ” bк°Җ мҲҳл ҙмқ„ м•Ҳ н•ңлӢӨлҠ” кІғ. 5,000лІҲ л°ҳліөн•ҙлҸ„ bлҠ” мөңм Ғк°’(-0.08)м—җ н•ңм°ё лӘ» лҜём№ҳлҠ” -0.0003м—җ лЁёл¬јл ҖлӢӨ. мӣҗмқёмқҖ мһ…л Ҙк°’мқҳ мҠӨмјҖмқј м°ЁмқҙмҳҖлӢӨ.

нҳ„мӢӨмқҖ мқҙліҙлӢӨ лҚ” ліөмһЎн•ҳлӢӨ. м§‘к°’мқҖ л©ҙм Ғ н•ҳлӮҳлЎң м •н•ҙм§Җм§Җ м•ҠлҠ”лӢӨ. л°© к°ңмҲҳ, мёөмҲҳ, м—ӯк№Ңм§Җ кұ°лҰ¬, кұҙм¶• м—°лҸ„вҖҰ ліҖмҲҳк°Җ мҲҳмӢӯ к°ңмқј мҲҳлҸ„ мһҲлӢӨ. ліҖмҲҳк°Җ лҠҳм–ҙлӮҳл©ҙ лӘЁлҚёмқҖ м–ҙл–»кІҢ л°”лҖҢкі , мҠӨмјҖмқј л¬ём ңлҠ” м–јл§ҲлӮҳ мӢ¬к°Ғн•ҙм§Ҳк№Ң?

лӢЁмқј ліҖмҲҳм—җм„ң лӢӨмӨ‘ ліҖмҲҳлЎң

м„ нҳ• нҡҢк·Җ кёҖм—җм„ң лӢӨлЈ¬ лӘЁлҚёмқҖ ліҖмҲҳк°Җ н•ҳлӮҳмҳҖлӢӨ.

Е· = wx + b

ліҖмҲҳк°Җ м—¬лҹ¬ к°ңл©ҙ к°Ғк°Ғм—җ к°ҖмӨ‘м№ҳлҘј л¶ҷмқёлӢӨ.

Е· = wвӮҒxвӮҒ + wвӮӮxвӮӮ + wвӮғxвӮғ + ... + wвӮҷxвӮҷ + b

мҳҲлҘј л“Өм–ҙ л©ҙм Ғ(xвӮҒ), л°© мҲҳ(xвӮӮ), мёөмҲҳ(xвӮғ)лЎң м§‘к°’мқ„ мҳҲмёЎн•ңлӢӨл©ҙ:

к°ҖкІ© = wвӮҒ Г— л©ҙм Ғ + wвӮӮ Г— л°©мҲҳ + wвӮғ Г— мёөмҲҳ + b

к°Ғ wбөўмқҳ мқҳлҜёлҠ” вҖңлӢӨлҘё ліҖмҲҳлҘј кі м •н–Ҳмқ„ л•Ң, xбөўк°Җ 1 мҰқк°Җн•ҳл©ҙ Е·к°Җ wбөўл§ҢнҒј ліҖн•ңлӢӨвҖқ лҠ” кІғмқҙлӢӨ. л©ҙм Ғмқҙ 1нҸү лҠҳм–ҙлӮ  л•Ң к°ҖкІ©мқҙ м–јл§ҲлӮҳ мҳӨлҘҙлҠ”м§Җ, л°©мқҙ н•ҳлӮҳ м¶”к°Җлҗ  л•Ң к°ҖкІ©мқҙ м–јл§ҲлӮҳ мҳӨлҘҙлҠ”м§ҖлҘј к°Ғк°Ғ лҸ…лҰҪм ҒмңјлЎң нҢҢм•…н•  мҲҳ мһҲлӢӨ.

лІЎн„°нҷ” н‘ңкё°

ліҖмҲҳк°Җ л§Һм•„м§Җл©ҙ мҲҳмӢқмқ„ мқјмқјмқҙ м“°кё° м–ҙл өлӢӨ. н–үл ¬лЎң н•ң лІҲм—җ н‘ңнҳ„н•ҳл©ҙ к№”лҒ”н•ҳлӢӨ.

лҚ°мқҙн„° н•ҳлӮҳлҘј лІЎн„°лЎң м“°л©ҙ:

x = [xвӮҒ, xвӮӮ, xвӮғ]    (нҠ№м„ұ лІЎн„°)
w = [wвӮҒ, wвӮӮ, wвӮғ]    (к°ҖмӨ‘м№ҳ лІЎн„°)

мҳҲмёЎк°’:

Е· = x В· w + b    (лӮҙм Ғ + нҺён–Ҙ)

лҚ°мқҙн„°к°Җ mк°ңл©ҙ н–үл ¬ X(m Г— n)лЎң н•ң лІҲм—җ кі„мӮ°н•ңлӢӨ.

Е· = Xw + b

мҪ”л“ңлЎң м“°л©ҙ y_pred = X @ w + b вҖ” н•ң мӨ„мқҙлӢӨ. NumPyмқҳ н–үл ¬ кіұ(@)мқҙ лӘЁл“  лҚ°мқҙн„° нҸ¬мқёнҠём—җ лҢҖн•ң мҳҲмёЎмқ„ лҸҷмӢңм—җ мІҳлҰ¬н•ңлӢӨ.

рҹ’Ў н‘ңкё°лІ• м •лҰ¬
n: нҠ№м„ұ(ліҖмҲҳ) мҲҳ, m: лҚ°мқҙн„° нҸ¬мқёнҠё мҲҳ.
XлҠ” (m Г— n) н–үл ¬ вҖ” н–үмқҙ лҚ°мқҙн„°, м—ҙмқҙ нҠ№м„ұ. wлҠ” (n,) лІЎн„°. мқҙ н‘ңкё°к°Җ scikit-learn, PyTorch л“ұ кұ°мқҳ лӘЁл“  ML лқјмқҙлёҢлҹ¬лҰ¬мқҳ н‘ңмӨҖмқҙлӢӨ.

лӢӨмӨ‘ м„ нҳ• нҡҢк·Җ кө¬нҳ„

мӢӨм ң лҚ°мқҙн„°лЎң кө¬нҳ„н•ҙліҙмһҗ. м•„нҢҢнҠё к°ҖкІ© мҳҲмёЎ вҖ” л©ҙм Ғ(нҸү), л°© мҲҳ, мёөмҲҳлҘј нҠ№м„ұмңјлЎң мӮ¬мҡ©н•ңлӢӨ.

import numpy as np

# м•„нҢҢнҠё лҚ°мқҙн„°: л©ҙм Ғ(нҸү), л°© мҲҳ, мёөмҲҳ вҶ’ к°ҖкІ©(м–өмӣҗ)
X = np.array([
    [142, 5, 23], [91, 2, 20], [132, 4,  3], [54, 2,  5],
    [146, 4, 19], [111, 5,  7], [100, 1, 21], [60, 4,  9],
    [142, 2,  7], [122, 5, 18], [126, 4,  4], [114, 1, 14],
    [114, 1, 18], [127, 3,  9], [156, 3, 21], [139, 2,  2],
    [143, 4, 20], [63, 4, 15], [42, 3,  7],  [61, 4, 12],
], dtype=float)

y = np.array([
    7.56, 4.63, 6.45, 3.34, 6.88, 6.80, 5.25, 4.84,
    5.97, 7.28, 6.06, 4.23, 5.01, 5.83, 6.70, 5.45,
    6.85, 4.73, 3.57, 4.84,
])

кІҪмӮ¬н•ҳк°•лІ•мқ„ лӢӨліҖмҲҳ лІ„м „мңјлЎң нҷ•мһҘн•ҳл©ҙ, н•өмӢ¬ ліҖнҷ”лҠ” dwк°Җ мҠӨм№јлқјм—җм„ң лІЎн„°к°Җ лҗңлӢӨлҠ” кІғмқҙлӢӨ.

# мҙҲкё°нҷ”
n_features = X.shape[1]  # 3
w = np.zeros(n_features)
b = 0.0
lr = 0.00001
epochs = 5000
m = len(y)

for epoch in range(epochs):
    y_pred = X @ w + b                    # (20,) = (20,3) @ (3,)
    error = y_pred - y                    # (20,)
    dw = (2/m) * (X.T @ error)            # (3,) = (3,20) @ (20,)
    db = (2/m) * np.sum(error)            # scalar
    w = w - lr * dw
    b = b - lr * db

    if epoch % 1000 == 0:
        cost = np.mean(error ** 2)
        print(f"Epoch {epoch:5d} | Cost: {cost:.4f} | w={np.round(w, 4)}")

мҷң? л©ҙм ҒмқҖ 42~156, л°© мҲҳлҠ” 1~5, мёөмҲҳлҠ” 2~23. к°’мқҳ лІ”мң„к°Җ м „нҳҖ лӢӨлҘҙлӢӨ.

Feature Scalingмқҙ н•„мҡ”н•ң мқҙмң 

л©ҙм Ғмқҳ лІ”мң„к°Җ 42~156мқҙкі  л°© мҲҳлҠ” 1~5лӢӨ. кё°мҡёкё°(gradient) кі„мӮ°м—җм„ң X.T @ errorлҘј н•ҳл©ҙ, л©ҙм Ғ м—ҙмқҖ к°’мқҙ нҒ¬лӢҲ кё°мҡёкё°лҸ„ нҒ¬кі , л°© мҲҳ м—ҙмқҖ к°’мқҙ мһ‘мңјлӢҲ кё°мҡёкё°лҸ„ мһ‘лӢӨ.

кІ°кіјм ҒмңјлЎң н•ҳлӮҳмқҳ н•ҷмҠөлҘ лЎң лӘЁл“  ліҖмҲҳлҘј лҸҷмӢңм—җ м Ғм ҲнһҲ м—…лҚ°мқҙнҠён•ҳлҠ” кІҢ л¶Ҳк°ҖлҠҘн•ҙм§„лӢӨ.

  • н•ҷмҠөлҘ мқ„ л©ҙм Ғм—җ л§һм¶”л©ҙ вҶ’ л°© мҲҳмҷҖ мёөмҲҳк°Җ л„Ҳл¬ҙ лҠҗлҰ¬кІҢ мҲҳл ҙ
  • н•ҷмҠөлҘ мқ„ л°© мҲҳм—җ л§һм¶”л©ҙ вҶ’ л©ҙм Ғмқҙ л°ңмӮ°

мқҙм „ кёҖм—җм„ң лӢЁмқј ліҖмҲҳмқёлҚ°лҸ„ мқҙ л¬ём ңлҘј кІӘм—ҲлӢӨ. л©ҙм Ғ(60~155)мқҙ нҒ¬лӢҲ wлҠ” л№ лҘҙкІҢ мҲҳл ҙн•ҳлҠ”лҚ° bлҠ” лҠҗл ёлӢӨ. ліҖмҲҳк°Җ м—¬лҹ¬ к°ңл©ҙ мқҙ л¬ём ңк°Җ нӣЁм”¬ мӢ¬к°Ғн•ҙм§„лӢӨ.

вҡ пёҸ л“ұкі м„ мңјлЎң мқҙн•ҙн•ҳкё°
л№„мҡ© н•ЁмҲҳлҘј 2D л“ұкі м„ мңјлЎң к·ёлҰ¬л©ҙ, мҠӨмјҖмқјмқҙ лӢӨлҘё ліҖмҲҳлҠ” к°ҖлҠҳкі  кёҙ нғҖмӣҗмқ„ л§Ңл“ лӢӨ. кІҪмӮ¬н•ҳк°•лІ•мқҖ мқҙ нғҖмӣҗмқҳ кёҙ м¶• л°©н–ҘмңјлЎң м§Җк·ёмһ¬к·ён•ҳл©° лҠҗлҰ¬кІҢ мҲҳл ҙн•ңлӢӨ. мҠӨмјҖмқјмқ„ л§һм¶”л©ҙ л“ұкі м„ мқҙ мӣҗм—җ к°Җк№ҢмӣҢм§Җкі , кІҪмӮ¬н•ҳк°•лІ•мқҖ кі§мһҘ мөңм Җм җмңјлЎң лӮҙл Өк°„лӢӨ.

н•ҙкІ°мұ…мқҖ к°„лӢЁн•ҳлӢӨ. лӘЁл“  ліҖмҲҳлҘј л№„мҠ·н•ң лІ”мң„лЎң л§һм¶°мӨҖлӢӨ. мқҙкІҢ Feature ScalingмқҙлӢӨ.

Feature Scaling л°©лІ• л№„көҗ

л°©лІ• кіөмӢқ кІ°кіј лІ”мң„ нҠ№м§•
Min-Max Normalization (x вҲ’ min) / (max вҲ’ min) [0, 1] лІ”мң„к°Җ лӘ…нҷ•, мқҙмғҒм№ҳм—җ лҜјк°җ
Standardization (Z-score) (x вҲ’ Ој) / Пғ нҸүк·  0, н‘ңмӨҖнҺём°Ё 1 мқҙмғҒм№ҳм—җ лҚң лҜјк°җ, к°ҖмһҘ лІ”мҡ©м Ғ
Mean Normalization (x вҲ’ Ој) / (max вҲ’ min) м•Ҫ [вҲ’0.5, 0.5] Min-MaxмҷҖ Standardizationмқҳ мӨ‘к°„

Min-Max Normalization

лӘЁл“  к°’мқ„ 0~1 мӮ¬мқҙлЎң м••м¶•н•ңлӢӨ.

X_min = X.min(axis=0)
X_max = X.max(axis=0)
X_minmax = (X - X_min) / (X_max - X_min)
# л©ҙм Ғ 42 вҶ’ 0.0, л©ҙм Ғ 156 вҶ’ 1.0

мһҘм җ: кІ°кіјк°Җ [0, 1]лЎң м§ҒкҙҖм Ғ. мқҙлҜём§Җ мІҳлҰ¬ л“ұ лІ”мң„к°Җ мӨ‘мҡ”н•ң кІҪмҡ° м Ғн•©. лӢЁм җ: мқҙмғҒм№ҳ(outlier)к°Җ н•ҳлӮҳлқјлҸ„ мһҲмңјл©ҙ лӮҳлЁём§Җ лҚ°мқҙн„°к°Җ мўҒмқҖ лІ”мң„лЎң лӘ°лҰ°лӢӨ.

Standardization (Z-score)

нҸүк· мқ„ 0, н‘ңмӨҖнҺём°ЁлҘј 1лЎң л§һм¶ҳлӢӨ. мӢӨл¬ҙм—җм„ң к°ҖмһҘ л§Һмқҙ м“°лҠ” л°©лІ•мқҙлӢӨ.

X_mean = X.mean(axis=0)  # [109.25, 3.15, 12.7]
X_std = X.std(axis=0)    # [34.58, 1.31, 6.79]
X_scaled = (X - X_mean) / X_std

мһҘм җ: мқҙмғҒм№ҳмқҳ мҳҒн–Ҙмқҙ мғҒлҢҖм ҒмңјлЎң мһ‘лӢӨ. лҢҖл¶Җл¶„мқҳ ML м•Ңкі лҰ¬мҰҳм—җм„ң кё°ліё м„ нғқ. лӢЁм җ: кІ°кіјк°Җ нҠ№м • лІ”мң„лЎң м ңн•ңлҗҳм§Җ м•ҠлҠ”лӢӨ (ліҙнҶө -3~3 м •лҸ„).

Mean Normalization

нҸүк· мқ„ л№јкі  лІ”мң„лЎң лӮҳлҲҲлӢӨ.

X_mean_norm = (X - X.mean(axis=0)) / (X.max(axis=0) - X.min(axis=0))

Min-MaxмҷҖ Standardization мӮ¬мқҙмқҳ м Ҳм¶©м•Ҳ. Andrew Ng к°•мқҳм—җм„ң мһҗмЈј л“ұмһҘн•ҳм§Җл§Ң, мӢӨл¬ҙм—җм„ңлҠ” Standardizationмқҙ лҚ” мқјл°ҳм ҒмқҙлӢӨ.

вң… м–ҙл–Ө л°©лІ•мқ„ м“ёк№Ң?
нҷ•мӢ мқҙ м—Ҷмңјл©ҙ Standardizationмқ„ м“ҙлӢӨ. м„ нҳ• нҡҢк·Җ, лЎңм§ҖмҠӨнӢұ нҡҢк·Җ, SVM, мӢ кІҪл§қ л“ұ кІҪмӮ¬н•ҳк°•лІ• кё°л°ҳ м•Ңкі лҰ¬мҰҳм—җм„ң кұ°мқҳ н•ӯмғҒ мһҳ лҸҷмһ‘н•ңлӢӨ. нҠёлҰ¬ кё°л°ҳ лӘЁлҚё(Decision Tree, Random Forest, XGBoost)мқҖ л¶„н•  кё°мӨҖмқҙ нҒ¬кё° л№„көҗлқјм„ң мҠӨмјҖмқјл§Ғмқҙ н•„мҡ” м—ҶлӢӨ.

Before/After: мҠӨмјҖмқјл§Ғмқҳ нҡЁкіј

к°ҷмқҖ лҚ°мқҙн„°, к°ҷмқҖ кІҪмӮ¬н•ҳк°•лІ•мқёлҚ° мҠӨмјҖмқјл§Ғ м—¬л¶Җл§Ң лӢӨлҘҙлӢӨ. м°Ёмқҙк°Җ к·№м ҒмқҙлӢӨ.

# Standardization м Ғмҡ©
X_mean = X.mean(axis=0)
X_std = X.std(axis=0)
X_scaled = (X - X_mean) / X_std

w = np.zeros(3)
b = 0.0
lr = 0.01  # н•ҷмҠөлҘ мқ„ 1,000л°° нӮӨмҡё мҲҳ мһҲлӢӨ!
m = len(y)

for epoch in range(5001):
    y_pred = X_scaled @ w + b
    error = y_pred - y
    cost = np.mean(error ** 2)
    dw = (2/m) * (X_scaled.T @ error)
    db = (2/m) * np.sum(error)
    w = w - lr * dw
    b = b - lr * db

    if epoch in [0, 10, 50, 100, 500, 1000]:
        print(f"Epoch {epoch:5d} | Cost: {cost:.4f}")

мҠӨмјҖмқјл§Ғ нӣ„ к°ҖмӨ‘м№ҳ н•ҙм„қ

мҠӨмјҖмқјл§Ғ нӣ„мқҳ к°ҖмӨ‘м№ҳ [0.856, 0.645, 0.178]мқҖ мӣҗлһҳ лӢЁмң„к°Җ м•„лӢҲлқј н‘ңмӨҖнҷ”лҗң лӢЁмң„лӢӨ. мӣҗлһҳ мҠӨмјҖмқјлЎң лҗҳлҸҢлҰ¬л Өл©ҙ:

# мҠӨмјҖмқјл§Ғлҗң к°ҖмӨ‘м№ҳ вҶ’ мӣҗлһҳ мҠӨмјҖмқј
w_original = w / X_std
b_original = b - np.sum(w * X_mean / X_std)
print(f"мӣҗлһҳ мҠӨмјҖмқј: w = {np.round(w_original, 4)}")
print(f"             b = {b_original:.4f}")
мӣҗлһҳ мҠӨмјҖмқј: w = [0.0247, 0.4909, 0.0262]
             b = 1.0321

н•ҙм„қн•ҳл©ҙ:

  • л©ҙм Ғ 1нҸү мҰқк°Җ вҶ’ к°ҖкІ© м•Ҫ 0.025м–ө(250л§Ңмӣҗ) мғҒмҠ№
  • л°© мҲҳ 1к°ң мҰқк°Җ вҶ’ к°ҖкІ© м•Ҫ 0.49м–ө(4,900л§Ңмӣҗ) мғҒмҠ№
  • мёөмҲҳ 1мёө мҰқк°Җ вҶ’ к°ҖкІ© м•Ҫ 0.026м–ө(260л§Ңмӣҗ) мғҒмҠ№

л°© мҲҳмқҳ мҳҒн–Ҙмқҙ л©ҙм ҒмқҙлӮҳ мёөмҲҳліҙлӢӨ нӣЁм”¬ нҒ¬лӢӨ. мҠӨмјҖмқјл§Ғ м „ к°ҖмӨ‘м№ҳ([0.04, 0.13, 0.05])лЎңлҠ” мқҙ мӮ¬мӢӨмқ„ м•Ң мҲҳ м—Ҷм—ҲлӢӨ.

sklearnмңјлЎң н•ң лІҲм—җ

мң„м—җм„ң м§Ғм ‘ кө¬нҳ„н•ң кұҙ мӣҗлҰ¬лҘј мқҙн•ҙн•ҳкё° мң„н•ҙм„ңмҳҖлӢӨ. мӢӨм „м—җм„ңлҠ” sklearnмқҳ PipelineмңјлЎң мҠӨмјҖмқјл§Ғкіј н•ҷмҠөмқ„ н•ң лІҲм—җ мІҳлҰ¬н•ңлӢӨ.

from sklearn.linear_model import LinearRegression
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline

# нҢҢмқҙн”„лқјмқё: мҠӨмјҖмқјл§Ғ вҶ’ м„ нҳ• нҡҢк·ҖлҘј н•ҳлӮҳлЎң
pipe = Pipeline([
    ('scaler', StandardScaler()),
    ('lr', LinearRegression())
])

pipe.fit(X, y)

print(f"RВІ = {pipe.score(X, y):.4f}")
print(f"coef = {pipe.named_steps['lr'].coef_}")
print(f"intercept = {pipe.named_steps['lr'].intercept_:.4f}")
RВІ = 0.9506
coef = [0.8555  0.6452  0.1777]
intercept = 5.6135

RВІ = 0.95 вҖ” м„ё к°Җм§Җ ліҖмҲҳлЎң к°ҖкІ© ліҖлҸҷмқҳ 95%лҘј м„ӨлӘ…н•ңлӢӨ. LinearRegressionмқҖ лӮҙл¶Җм ҒмңјлЎң м •к·ң л°©м •мӢқмқ„ м“°кё° л•Ңл¬ём—җ мҠӨмјҖмқјл§Ғ м—ҶмқҙлҸ„ м •нҷ•н•ң н•ҙлҘј кө¬н•ҳм§Җл§Ң, нҢҢмқҙн”„лқјмқём—җ л„Јм–ҙл‘җл©ҙ лӢӨлҘё лӘЁлҚёлЎң көҗмІҙн•  л•Ң(лЎңм§ҖмҠӨнӢұ нҡҢк·Җ, SVM л“ұ) мҠӨмјҖмқјл§Ғмқҙ мқҙлҜё м Ғмҡ©лҗҳм–ҙ мһҲм–ҙ нҺён•ҳлӢӨ.

# мғҲ м•„нҢҢнҠё мҳҲмёЎ
import numpy as np
new_house = np.array([[85, 3, 10]])  # л©ҙм Ғ 85нҸү, л°© 3к°ң, 10мёө
pred = pipe.predict(new_house)
print(f"мҳҲмёЎ к°ҖкІ©: {pred[0]:.2f}м–өмӣҗ")  # вҶ’ 4.87м–өмӣҗ
рҹ’Ў Pipelineмқ„ м“°лҠ” м§„м§ң мқҙмң 
мҠӨмјҖмқјл§Ғкіј лӘЁлҚёмқ„ л”°лЎң кҙҖлҰ¬н•ҳл©ҙ, мғҲ лҚ°мқҙн„°лҘј мҳҲмёЎн•  л•Ң мҠӨмјҖмқјл§Ғмқ„ мһҠлҠ” мӢӨмҲҳк°Җ мғқкёҙлӢӨ. PipelineмқҖ fit, predict, scoreлҘј нҳём¶ңн•ҳл©ҙ м „мІҳлҰ¬мҷҖ лӘЁлҚёмқ„ мһҗлҸҷмңјлЎң мҲңм„ңлҢҖлЎң м Ғмҡ©н•ңлӢӨ. мҪ”л“ңк°Җ к№”лҒ”н•ҙм§Җкі  мӢӨмҲҳк°Җ мӨ„м–ҙл“ лӢӨ.

нқ”н•ң мӢӨмҲҳ

1. Test лҚ°мқҙн„°м—җ Trainмқҳ мҠӨмјҖмқјлҹ¬лҘј м Ғмҡ©н•ҳм§Җ м•ҠлҠ”лӢӨ

# вқҢ мһҳлӘ»лҗң л°©лІ•: test лҚ°мқҙн„°лҘј л”°лЎң мҠӨмјҖмқјл§Ғ
scaler_train = StandardScaler().fit(X_train)
X_train_scaled = scaler_train.transform(X_train)

scaler_test = StandardScaler().fit(X_test)  # лі„лҸ„ fit вҶ’ кё°мӨҖмқҙ лӢӨлҰ„!
X_test_scaled = scaler_test.transform(X_test)
# вң… мҳ¬л°”лҘё л°©лІ•: trainмңјлЎң fitн•ң scalerлҘј testм—җлҸ„ м Ғмҡ©
scaler = StandardScaler().fit(X_train)
X_train_scaled = scaler.transform(X_train)
X_test_scaled = scaler.transform(X_test)  # к°ҷмқҖ scalerлЎң transformл§Ң

Train лҚ°мқҙн„°мқҳ нҸүк· кіј н‘ңмӨҖнҺём°ЁлЎң test лҚ°мқҙн„°лҘј ліҖнҷҳн•ҙм•ј н•ңлӢӨ. Test лҚ°мқҙн„°лЎң лі„лҸ„лЎң fitн•ҳл©ҙ кё°мӨҖмқҙ лӢ¬лқјм ём„ң лӘЁлҚёмқҙ м—үлҡұн•ң мҳҲмёЎмқ„ н•ңлӢӨ. Pipelineмқ„ м“°л©ҙ мқҙ мӢӨмҲҳлҘј кө¬мЎ°м ҒмңјлЎң л°©м§Җн•  мҲҳ мһҲлӢӨ.

2. лІ”мЈјнҳ• ліҖмҲҳм—җ мҠӨмјҖмқјл§Ғмқ„ м Ғмҡ©н•ңлӢӨ

# вқҢ мӣҗ-н•« мқёмҪ”л”©лҗң ліҖмҲҳк№Ңм§Җ мҠӨмјҖмқјл§Ғ
# gender: [0, 1], type: [0, 0, 1] вҶ’ мқҙлҜё лІ”мң„к°Җ м •н•ҙм ё мһҲлӢӨ

мӣҗ-н•« мқёмҪ”л”©мқҙлӮҳ 0/1 мқҙм§„ ліҖмҲҳлҠ” мҠӨмјҖмқјл§Ғн•  н•„мҡ”к°Җ м—ҶлӢӨ. мҳӨнһҲл Ө мҠӨмјҖмқјл§Ғн•ҳл©ҙ мқҳлҜёк°Җ мҷңкіЎлҗңлӢӨ. мҲҳм№ҳнҳ• м—°мҶҚ ліҖмҲҳл§Ң мҠӨмјҖмқјл§Ғн•ңлӢӨ.

3. нғҖкІҹ(y)к№Ңм§Җ мҠӨмјҖмқјл§Ғн•ңлӢӨ

мқјл°ҳм Ғмқё м„ нҳ• нҡҢк·Җм—җм„ң yлҠ” мҠӨмјҖмқјл§Ғн•ҳм§Җ м•ҠлҠ”лӢӨ. yлҘј мҠӨмјҖмқјл§Ғн•ҳл©ҙ мҳҲмёЎк°’лҸ„ мҠӨмјҖмқјл§Ғлҗң лӢЁмң„лЎң лӮҳмҳӨкё° л•Ңл¬ём—җ лӢӨмӢң м—ӯліҖнҷҳн•ҙм•ј н•ҳкі , н•ҙм„қмқҙ ліөмһЎн•ҙм§„лӢӨ. лӢЁ, мӢ кІҪл§қм—җм„ң yмқҳ лІ”мң„к°Җ к·№лӢЁм ҒмңјлЎң нҒ¬л©ҙ н•ҷмҠө м•Ҳм •м„ұмқ„ мң„н•ҙ мҠӨмјҖмқјл§Ғн•ҳкё°лҸ„ н•ңлӢӨ вҖ” мқҙкұҙ нҠ№мҲҳн•ң кІҪмҡ°лӢӨ.

вҡ пёҸ лҚ°мқҙн„° лҲ„мҲҳ(Data Leakage) мЈјмқҳ
Train/Test л¶„н•  м „м—җ м „мІҙ лҚ°мқҙн„°лЎң мҠӨмјҖмқјл§Ғн•ҳл©ҙ, test лҚ°мқҙн„°мқҳ нҶөкі„к°Җ trainм—җ м„һм—¬ л“Өм–ҙк°„лӢӨ. л°ҳл“ңмӢң л¶„н•  нӣ„ train лҚ°мқҙн„°л§ҢмңјлЎң fitн•ҳкі , к·ё кё°мӨҖмңјлЎң testлҘј transformн•ҙм•ј н•ңлӢӨ.

л§Ҳм№ҳл©°

ліҖмҲҳк°Җ м—¬лҹ¬ к°ңк°Җ лҗҳл©ҙ лӘЁлҚё мһҗмІҙліҙлӢӨ м „мІҳлҰ¬к°Җ кІ°кіјлҘј мўҢмҡ°н•ңлӢӨ. к°ҷмқҖ кІҪмӮ¬н•ҳк°•лІ•, к°ҷмқҖ лҚ°мқҙн„°мқёлҚ° Feature Scaling н•ҳлӮҳлЎң 5,000лІҲ лҸҢл ӨлҸ„ м•Ҳ лҗҳлҚҳ мҲҳл ҙмқҙ 268лІҲ л§Ңм—җ лҒқлӮңлӢӨ. мқҙкұҙ мқҙлЎ м Ғ м°Ёмқҙк°Җ м•„лӢҲлқј мӢӨм „м—җм„ң л§ӨлІҲ л§ҲмЈјн•ҳлҠ” м°ЁмқҙлӢӨ.

м—¬кё°к№Ңм§Җк°Җ нҡҢк·Җ(Regression) мқҳ мқҙм•јкё°лӢӨ. лӢӨмқҢ кёҖм—җм„ңлҠ” м—°мҶҚк°’ мҳҲмёЎмқҙ м•„лӢҢ, вҖңYes лҳҗлҠ” NoвҖқлҘј нҢҗлӢЁн•ҳлҠ” л¶„лҘҳ(Classification) л¬ём ңлЎң л„ҳм–ҙк°„лӢӨ. м„ нҳ• нҡҢк·Җм—җ мӢңк·ёлӘЁмқҙл“ң н•ЁмҲҳ н•ҳлӮҳлҘј м–№м–ҙ нҷ•лҘ мқ„ м¶ңл Ҙн•ҳлҠ” лЎңм§ҖмҠӨнӢұ нҡҢк·Җ(Logistic Regression) лҘј лӢӨлЈ¬лӢӨ.
