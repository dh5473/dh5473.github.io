лЎңм§ҖмҠӨнӢұ нҡҢк·ҖлҠ” нҷ•лҘ лЎң л¶„лҘҳн•ҳкі , KNNмқҖ кұ°лҰ¬лЎң л¶„лҘҳн•ңлӢӨ. SVMлҸ„ кұ°лҰ¬лҘј мӮ¬мҡ©н•ҳм§Җл§Ң, л°ңмғҒмқҙ к·јліём ҒмңјлЎң лӢӨлҘҙлӢӨ. KNNмқҙ вҖңк°ҖмһҘ к°Җк№Ңмҡҙ мқҙмӣғмқҙ лӯҗлғҗвҖқлҘј л¬»лҠ”лӢӨл©ҙ, SVMмқҖ вҖңл‘җ нҒҙлһҳмҠӨ мӮ¬мқҙм—җ к°ҖмһҘ л„“мқҖ лҸ„лЎңлҘј к№” мҲҳ мһҲлҠ” кІҪкі„лҠ” м–ҙл””лғҗвҖқ лҘј л¬»лҠ”лӢӨ.


мқҙ вҖңлҸ„лЎңмқҳ нҸӯвҖқмқ„ л§Ҳм§„(Margin) мқҙлқј л¶ҖлҘҙкі , л§Ҳм§„мқҙ мөңлҢҖмқё кІҪкі„лҘј м°ҫлҠ” кІғмқҙ SVMмқҳ н•өмӢ¬мқҙлӢӨ. м§ҒкҙҖм ҒмңјлЎңлҸ„ лҸ„лЎңк°Җ л„“мқ„мҲҳлЎқ мғҲ лҚ°мқҙн„°к°Җ л“Өм–ҙмҷ”мқ„ л•Ң мҳ¬л°”лҘё мӘҪм—җ л–Ём–ҙм§Ҳ к°ҖлҠҘм„ұмқҙ лҶ’лӢӨ. мқҙ кёҖм—җм„ңлҠ” мөңлҢҖ л§Ҳм§„мқҳ кё°н•ҳн•ҷм Ғ мқҳлҜёл¶Җн„° мӢңмһ‘н•ҙ, мҶҢн”„нҠё л§Ҳм§„(C нҢҢлқјлҜён„°), м»Өл„җ нҠёлҰӯ(RBF), к·ёлҰ¬кі  sklearn мӢӨм „ нҢҢмқҙн”„лқјмқёк№Ңм§Җ лӢӨлЈ¬лӢӨ.

л§Ҳм§„ кІҪкі„м—җ л”ұ кұёміҗ мһҲлҠ” лҚ°мқҙн„° нҸ¬мқёнҠёл“Өмқҙ м„ңнҸ¬нҠё лІЎн„°(Support Vector) лӢӨ. мқҙлҰ„ к·ёлҢҖлЎң кІ°м • кІҪкі„лҘј вҖңм§Җнғұн•ҳлҠ”(support)вҖқ лІЎн„°л“ӨмқҙлӢӨ. мӮ¬мӢӨ кІ°м • кІҪкі„мқҳ мң„м№ҳлҠ” м„ңнҸ¬нҠё лІЎн„°л§ҢмңјлЎң кІ°м •лҗҳкі , лӮҳлЁём§Җ лҚ°мқҙн„°лҠ” м•„л¬ҙлҰ¬ л§Һм•„лҸ„ кІҪкі„м—җ мҳҒн–Ҙмқ„ мЈјм§Җ м•ҠлҠ”лӢӨ. мқҙ м„ұм§Ҳмқҙ SVMмқ„ лҸ…нҠ№н•ҳкІҢ л§Ңл“ңлҠ” н•өмӢ¬мқҙлӢӨ.

рҹ’Ў мҷң л§Ҳм§„мқҙ нҒҙмҲҳлЎқ мўӢмқ„к№Ң?
л§Ҳм§„мқҙ л„“лӢӨлҠ” кұҙ, кІ°м • кІҪкі„к°Җ м–‘мӘҪ нҒҙлһҳмҠӨлЎңл¶Җн„° м¶©л¶„нһҲ л–Ём–ҙм ё мһҲлӢӨлҠ” лң»мқҙлӢӨ. мғҲ лҚ°мқҙн„°м—җ м•Ҫк°„мқҳ л…ёмқҙмҰҲк°Җ м„һм—¬ мһҲм–ҙлҸ„ мҳ¬л°”лҘё мӘҪм—җ л–Ём–ҙм§Ҳ м—¬мң к°Җ мһҲлӢӨ. нҶөкі„ н•ҷмҠө мқҙлЎ м—җм„ңлҠ” мқҙлҘј кө¬мЎ°м Ғ мң„н—ҳ мөңмҶҢнҷ”(Structural Risk Minimization)лқјкі  л¶ҖлҘёлӢӨ вҖ” л§Ҳм§„мқ„ мөңлҢҖнҷ”н•ҳл©ҙ VC м°Ёмӣҗмқҙ м ңн•ңлҗҳм–ҙ мқјл°ҳнҷ” мҳӨм°Ёмқҳ мғҒн•ңмқҙ лӮ®м•„м§„лӢӨ.

мҶҢн”„нҠё л§Ҳм§„: нҳ„мӢӨмқҳ лҚ°мқҙн„°лҠ” к№”лҒ”н•ҳм§Җ м•ҠлӢӨ

мң„мқҳ м„ӨлӘ…мқҖ н•ҳл“ң л§Ҳм§„(Hard Margin) вҖ” л‘җ нҒҙлһҳмҠӨк°Җ мҷ„лІҪнһҲ л¶„лҰ¬ к°ҖлҠҘн•ң кІҪмҡ°м—җл§Ң м„ұлҰҪн•ңлӢӨ. нҳ„мӢӨ лҚ°мқҙн„°м—җм„ңлҠ” нҒҙлһҳмҠӨк°Җ кІ№м№ҳкұ°лӮҳ, мқҙмғҒм№ҳк°Җ м„һм—¬ мһҲм–ҙм„ң к№”лҒ”н•ң л¶„лҰ¬к°Җ л¶Ҳк°ҖлҠҘн•ң кІҪмҡ°к°Җ лҢҖл¶Җл¶„мқҙлӢӨ.

н•ҳл“ң л§Ҳм§„ SVMмқ„ мқҙлҹ° лҚ°мқҙн„°м—җ м Ғмҡ©н•ҳл©ҙ л‘җ к°Җм§Җ мӨ‘ н•ҳлӮҳк°Җ л°ңмғқн•ңлӢӨ:

  • н•ҙк°Җ мЎҙмһ¬н•ҳм§Җ м•ҠлҠ”лӢӨ (л¶„лҰ¬ л¶Ҳк°Җ)
  • мқҙмғҒм№ҳ н•ҳлӮҳм—җ кІҪкі„к°Җ к·№лӢЁм ҒмңјлЎң мҷңкіЎлҗңлӢӨ

мҶҢн”„нҠё л§Ҳм§„(Soft Margin) SVMмқҖ мқјл¶Җ лҚ°мқҙн„°к°Җ л§Ҳм§„ м•Ҳм—җ л“Өм–ҙмҳӨкұ°лӮҳ, мӢ¬м§Җм–ҙ л°ҳлҢҖмӘҪмңјлЎң л„ҳм–ҙк°ҖлҠ” кІғмқ„ н—Ҳмҡ©н•ңлӢӨ. лҢҖмӢ  к·ё мң„л°ҳм—җ лҢҖн•ҙ лІҢм җ(penalty) мқ„ л¶Җкіјн•ңлӢӨ.

к·ңм ң(Regularization) кёҖм—җм„ң лӢӨлӨҳлҚҳ alpha(lambda)мҷҖ м •нҷ•нһҲ м—ӯмҲҳ кҙҖкі„лӢӨ. лЎңм§ҖмҠӨнӢұ нҡҢк·Җмқҳ LogisticRegression(C=1.0)лҸ„ к°ҷмқҖ мӣҗлҰ¬мҳҖлӢӨ вҖ” CлҠ” к·ңм ң к°•лҸ„мқҳ м—ӯмҲҳлӢӨ.

sklearnмңјлЎң Cм—җ л”°лҘё ліҖнҷ”лҘј нҷ•мқён•ҙліҙмһҗ.

from sklearn.svm import SVC
from sklearn.datasets import make_moons
from sklearn.model_selection import cross_val_score
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline

X, y = make_moons(n_samples=200, noise=0.3, random_state=42)

for C in [0.01, 0.1, 1, 10, 100]:
    pipe = Pipeline([
        ('scaler', StandardScaler()),
        ('svc', SVC(kernel='rbf', C=C))
    ])
    scores = cross_val_score(pipe, X, y, cv=5)
    print(f"C={C:6.2f} | м •нҷ•лҸ„: {scores.mean():.3f} (Вұ{scores.std():.3f})")

C=  0.10 | м •нҷ•лҸ„: 0.875 (Вұ0.048)
C=  1.00 | м •нҷ•лҸ„: 0.900 (Вұ0.032)
C= 10.00 | м •нҷ•лҸ„: 0.895 (Вұ0.037)
C=100.00 | м •нҷ•лҸ„: 0.890 (Вұ0.040)

м»Өл„җ нҠёлҰӯ: л№„м„ нҳ• л¶„лҘҳ

м—¬кё°к№Ңм§Җмқҳ SVMмқҖ м§Ғм„ (лҳҗлҠ” мҙҲнҸүл©ҙ)мңјлЎң л¶„лҘҳн–ҲлӢӨ. к·ёлҹ°лҚ° лҚ°мқҙн„°к°Җ мӣҗнҳ•мңјлЎң л¶„нҸ¬н•ҳкұ°лӮҳ, XOR нҢЁн„ҙмІҳлҹј м§Ғм„ мңјлЎңлҠ” м ҲлҢҖ л¶„лҰ¬н•  мҲҳ м—ҶлҠ” кІҪмҡ°к°Җ мһҲлӢӨ.

                                 в—Ӣ   в—Ӣ   в—Ӣ

2м°Ёмӣҗм—җм„ң мӣҗ м•Ҳм—җ мһҲлҠ” нҒҙлһҳмҠӨ(в—Ҹ)мҷҖ л°–м—җ мһҲлҠ” нҒҙлһҳмҠӨ(в—Ӣ)лҠ” м§Ғм„ мңјлЎң л¶„лҰ¬н•  мҲҳ м—ҶлӢӨ. н•ҳм§Җл§Ң ПҶ(xвӮҒ, xвӮӮ) = (xвӮҒ, xвӮӮ, xвӮҒВІ + xвӮӮВІ)мІҳлҹј мғҲ м°Ёмӣҗмқ„ м¶”к°Җн•ҳл©ҙ, 3м°Ёмӣҗм—җм„ң нҸүл©ҙ н•ҳлӮҳлЎң к№”лҒ”н•ҳкІҢ л¶„лҰ¬лҗңлӢӨ.

л¬ём ңлҠ” кі м°Ёмӣҗ ліҖнҷҳмқҳ л№„мҡ©мқҙлӢӨ. ліҖмҲҳк°Җ dк°ңмқҙкі  2м°Ё лӢӨн•ӯмӢқмңјлЎң ліҖнҷҳн•ҳл©ҙ O(dВІ)к°ңмқҳ мғҲ ліҖмҲҳк°Җ мғқкё°кі , кі м°ЁлЎң к°ҲмҲҳлЎқ кё°н•ҳкёүмҲҳм ҒмңјлЎң лҠҳм–ҙлӮңлӢӨ.

м»Өл„җ нҠёлҰӯмқҙлһҖ

м»Өл„җ нҠёлҰӯ(Kernel Trick) мқҖ мқҙ л¬ём ңлҘј мҡ°нҡҢн•ңлӢӨ. SVMмқҳ мөңм Ғнҷ” л¬ём ңлҘј мһҗм„ёнһҲ ліҙл©ҙ, лҚ°мқҙн„° нҸ¬мқёнҠё к°„мқҳ лӮҙм Ғ(dot product) л§ҢмңјлЎң кі„мӮ°мқҙ к°ҖлҠҘн•ҳлӢӨ. м»Өл„җ н•ЁмҲҳ K(xбөў, xвұј)лҠ” кі м°Ёмӣҗм—җм„ңмқҳ лӮҙм Ғмқ„ м Җм°Ёмӣҗм—җм„ң м§Ғм ‘ кі„мӮ°н•ңлӢӨ вҖ” мӢӨм ңлЎң ліҖнҷҳн•  н•„мҡ”к°Җ м—ҶлӢӨ.

м»Өл„җ мҲҳмӢқ нҠ№м§•
Linear K(x, xвҖҷ) = x В· xвҖҷ м„ нҳ• л¶„лҰ¬ к°ҖлҠҘн•  л•Ң
Polynomial K(x, xвҖҷ) = (Оіx В· xвҖҷ + r)^d лӢӨн•ӯмӢқ кІҪкі„
RBF (Gaussian) K(x, xвҖҷ) = exp(-Оі||x - xвҖҷ||ВІ) к°ҖмһҘ лІ”мҡ©м Ғ, л¬ҙн•ң м°Ёмӣҗ л§Өн•‘

мӢӨм „м—җм„ң к°ҖмһҘ л§Һмқҙ м“°мқҙлҠ” кұҙ RBF(Radial Basis Function) м»Өл„җмқҙлӢӨ. л‘җ лҚ°мқҙн„° нҸ¬мқёнҠё мӮ¬мқҙмқҳ мң нҒҙлҰ¬л“ң кұ°лҰ¬лҘј кё°л°ҳмңјлЎң мң мӮ¬лҸ„лҘј кі„мӮ°н•ҳл©°, мқҙлЎ м ҒмңјлЎң л¬ҙн•ң м°ЁмӣҗмңјлЎңмқҳ л§Өн•‘м—җ н•ҙлӢ№н•ңлӢӨ.

gamma нҢҢлқјлҜён„°

RBF м»Өл„җмқҳ gamma(Оі) лҠ” к°Ғ лҚ°мқҙн„° нҸ¬мқёнҠёмқҳ мҳҒн–Ҙ лІ”мң„лҘј кІ°м •н•ңлӢӨ.

  • gammaк°Җ мһ‘мңјл©ҙ: мҳҒн–Ҙ лІ”мң„к°Җ л„“лӢӨ вҶ’ л¶Җл“ңлҹ¬мҡҙ кІ°м • кІҪкі„ вҶ’ кіјмҶҢм Ғн•© кІҪн–Ҙ
  • gammaк°Җ нҒ¬л©ҙ: мҳҒн–Ҙ лІ”мң„к°Җ мўҒлӢӨ вҶ’ к°Ғ нҸ¬мқёнҠё мЈјліҖл§Ң мҳҒн–Ҙ вҶ’ ліөмһЎн•ң кІҪкі„ вҶ’ кіјм Ғн•© кІҪн–Ҙ

Linear м»Өл„җкіј RBF м»Өл„җмқ„ л№„көҗн•ҙліҙмһҗ.

import numpy as np
from sklearn.svm import SVC
from sklearn.datasets import make_moons
from sklearn.model_selection import cross_val_score
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline

X, y = make_moons(n_samples=300, noise=0.25, random_state=42)

kernels = {
    'Linear': SVC(kernel='linear', C=1),
    'RBF (gamma=0.1)': SVC(kernel='rbf', C=1, gamma=0.1),
    'RBF (gamma=1)': SVC(kernel='rbf', C=1, gamma=1),
    'RBF (gamma=10)': SVC(kernel='rbf', C=1, gamma=10),
}

for name, svc in kernels.items():
    pipe = Pipeline([('scaler', StandardScaler()), ('svc', svc)])
    scores = cross_val_score(pipe, X, y, cv=5)
    print(f"{name:20s} | м •нҷ•лҸ„: {scores.mean():.3f} (Вұ{scores.std():.3f})")
Linear               | м •нҷ•лҸ„: 0.873 (Вұ0.025)
RBF (gamma=0.1)      | м •нҷ•лҸ„: 0.887 (Вұ0.031)
RBF (gamma=1)        | м •нҷ•лҸ„: 0.907 (Вұ0.022)
RBF (gamma=10)       | м •нҷ•лҸ„: 0.890 (Вұ0.035)

Linear м»Өл„җлЎңлҠ” л°ҳлӢ¬ лӘЁм–‘ лҚ°мқҙн„°лҘј м ңлҢҖлЎң л¶„лҰ¬н•ҳм§Җ лӘ»н•ңлӢӨ. RBF м»Өл„җмқ„ м“°л©ҙ л№„м„ нҳ• кІҪкі„к°Җ к°ҖлҠҘн•ҙм§Җл©ҙм„ң м •нҷ•лҸ„к°Җ мҳ¬лқјк°„лӢӨ. н•ҳм§Җл§Ң gamma=10мІҳлҹј л„Ҳл¬ҙ лҶ’мқҙл©ҙ к°Ғ лҚ°мқҙн„° нҸ¬мқёнҠём—җ кіјн•ҳкІҢ л§һм¶”л©ҙм„ң лӢӨмӢң м„ұлҠҘмқҙ л–Ём–ҙм§„лӢӨ.

вҡ пёҸ CмҷҖ gammaлҠ” н•Ёк»ҳ нҠңлӢқн•ҙм•ј н•ңлӢӨ
CлҠ” мҳӨл¶„лҘҳ н—Ҳмҡ© м •лҸ„лҘј, gammaлҠ” кІ°м • кІҪкі„мқҳ ліөмһЎлҸ„лҘј м ңм–ҙн•ңлӢӨ. л‘ҳ лӢӨ лҶ’мңјл©ҙ к·№лӢЁм Ғ кіјм Ғн•©, л‘ҳ лӢӨ лӮ®мңјл©ҙ кіјмҶҢм Ғн•©мқҙ лҗңлӢӨ. GridSearchCVлЎң л‘җ нҢҢлқјлҜён„°лҘј лҸҷмӢңм—җ нғҗмғүн•ҳлҠ” кІғмқҙ н•өмӢ¬мқҙлӢӨ.

sklearn мӢӨм „ мҪ”л“ң

Breast Cancer лҚ°мқҙн„°м…ӢмңјлЎң SVM нҢҢмқҙн”„лқјмқёмқ„ кө¬м„ұн•ҙліҙмһҗ. м „мІҳлҰ¬(мҠӨмјҖмқјл§Ғ), лӘЁлҚё нӣҲл Ё, н•ҳмқҙнҚјнҢҢлқјлҜён„° нҠңлӢқк№Ңм§Җ н•ң лІҲм—җ м§„н–үн•ңлӢӨ.

import numpy as np
from sklearn.datasets import load_breast_cancer
from sklearn.model_selection import train_test_split, GridSearchCV
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline
from sklearn.svm import SVC
from sklearn.metrics import accuracy_score, classification_report

# лҚ°мқҙн„° мӨҖл№„
data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.2, random_state=42
)

# нҢҢмқҙн”„лқјмқё: StandardScaler вҶ’ SVC
pipe = Pipeline([
    ('scaler', StandardScaler()),
    ('svc', SVC())
])

# GridSearchCVлЎң C, gamma лҸҷмӢң нғҗмғү
param_grid = {
    'svc__C': [0.1, 1, 10, 100],
    'svc__gamma': ['scale', 0.01, 0.1, 1],
    'svc__kernel': ['rbf']
}

grid = GridSearchCV(pipe, param_grid, cv=5, scoring='accuracy', n_jobs=-1)
grid.fit(X_train, y_train)

print(f"мөңм Ғ нҢҢлқјлҜён„°: {grid.best_params_}")
print(f"CV м •нҷ•лҸ„: {grid.best_score_:.4f}")
print(f"н…ҢмҠӨнҠё м •нҷ•лҸ„: {grid.score(X_test, y_test):.4f}")
мөңм Ғ нҢҢлқјлҜён„°: {'svc__C': 10, 'svc__gamma': 0.01, 'svc__kernel': 'rbf'}
CV м •нҷ•лҸ„: 0.9780
н…ҢмҠӨнҠё м •нҷ•лҸ„: 0.9825

30к°ң нҠ№м„ұм—җм„ң C=10, gamma=0.01мқҙ мөңм ҒмңјлЎң м„ нғқлҗҳм—ҲлӢӨ. 98.25%мқҳ н…ҢмҠӨнҠё м •нҷ•лҸ„лӢӨ.

вң… SVMм—җлҠ” л°ҳл“ңмӢң мҠӨмјҖмқјл§Ғмқҙ н•„мҡ”н•ҳлӢӨ
SVMмқҖ лҚ°мқҙн„° нҸ¬мқёнҠё к°„ кұ°лҰ¬лҘј кё°л°ҳмңјлЎң мһ‘лҸҷн•ңлӢӨ. ліҖмҲҳмқҳ мҠӨмјҖмқјмқҙ лӢӨлҘҙл©ҙ кұ°лҰ¬ кі„мӮ°мқҙ мҷңкіЎлҗңлӢӨ. Pipelineм—җ StandardScalerлҘј л„ЈлҠ” кұҙ м„ нғқмқҙ м•„лӢҲлқј н•„мҲҳлӢӨ.

лЎңм§ҖмҠӨнӢұ нҡҢк·Җ, KNNкіј л№„көҗ

к°ҷмқҖ лҚ°мқҙн„°м—җм„ң м„ё лӘЁлҚёмқҳ м„ұлҠҘмқ„ л№„көҗн•ҙліҙмһҗ.

from sklearn.linear_model import LogisticRegression
from sklearn.neighbors import KNeighborsClassifier

models = {
    'Logistic Regression': Pipeline([
        ('scaler', StandardScaler()),
        ('model', LogisticRegression(C=1, max_iter=1000))
    ]),
    'KNN (k=5)': Pipeline([
        ('scaler', StandardScaler()),
        ('model', KNeighborsClassifier(n_neighbors=5))
    ]),
    'SVM (RBF)': Pipeline([
        ('scaler', StandardScaler()),
        ('model', SVC(C=10, gamma=0.01, kernel='rbf'))
    ]),
}

for name, pipe in models.items():
    pipe.fit(X_train, y_train)
    train_acc = pipe.score(X_train, y_train)
    test_acc = pipe.score(X_test, y_test)
    print(f"{name:25s} | нӣҲл Ё: {train_acc:.4f} | н…ҢмҠӨнҠё: {test_acc:.4f}")

KNN (k=5)                 | нӣҲл Ё: 0.9736 | н…ҢмҠӨнҠё: 0.9649
SVM (RBF)                 | нӣҲл Ё: 0.9912 | н…ҢмҠӨнҠё: 0.9825

мқҙ лҚ°мқҙн„°м…Ӣм—җм„ңлҠ” SVMмқҙ к°ҖмһҘ лҶ’мқҖ н…ҢмҠӨнҠё м •нҷ•лҸ„лҘј ліҙмқёлӢӨ. л¬јлЎ  лҚ°мқҙн„°м—җ л”°лқј кІ°кіјлҠ” лӢ¬лқјм§„лӢӨ вҖ” н•өмӢ¬мқҖ лӘЁлҚёл§ҲлӢӨ к°•м җмқҙ лӢӨлҘҙлӢӨлҠ” кІғмқҙлӢӨ.

SVMмқҳ мһҘлӢЁм җкіј м„ нғқ кё°мӨҖ

л¶„лҘҳ лӘЁлҚё л№„көҗ

Logistic Regression KNN SVM
л¶„лҘҳ л°©мӢқ нҷ•лҘ  (мӢңк·ёлӘЁмқҙл“ң) мөңк·јм ‘ мқҙмӣғ кұ°лҰ¬ мөңлҢҖ л§Ҳм§„ мҙҲнҸүл©ҙ
кІ°м • кІҪкі„ м„ нҳ• (кё°ліё) л№„м„ нҳ• (мһҗлҸҷ) м„ нҳ•/л№„м„ нҳ• (м»Өл„җ)
н•ҷмҠө мҶҚлҸ„ л№ лҰ„ н•ҷмҠө м—ҶмқҢ (lazy) лҠҗлҰј (O(nВІ~nВі))
мҳҲмёЎ мҶҚлҸ„ л№ лҰ„ лҠҗлҰј (м „мІҙ нғҗмғү) л№ лҰ„ (м„ңнҸ¬нҠё лІЎн„°л§Ң)
нҷ•лҘ  м¶ңл Ҙ л„ӨмқҙнӢ°лёҢ к°ҖлҠҘ Platt scaling н•„мҡ”
мҠӨмјҖмқјл§Ғ к¶ҢмһҘ н•„мҲҳ н•„мҲҳ
кі м°Ёмӣҗ лҚ°мқҙн„° ліҙнҶө м°Ёмӣҗмқҳ м ҖмЈј к°•н•Ё

мһҘм җ

SVMмқҙ л№ӣлӮҳлҠ” мҲңк°„л“Өмқҙ мһҲлӢӨ:

  • кі м°Ёмӣҗ лҚ°мқҙн„°м—җ к°•н•ҳлӢӨ: н…ҚмҠӨнҠё л¶„лҘҳмІҳлҹј ліҖмҲҳк°Җ мҲҳмІң~мҲҳл§Ң к°ңмқё кІҪмҡ°, SVMмқҖ л§Ҳм§„ мөңлҢҖнҷ” лҚ•л¶„м—җ м°Ёмӣҗмқҳ м ҖмЈјм—җ мғҒлҢҖм ҒмңјлЎң к°•н•ҳлӢӨ
  • мқјл°ҳнҷ” м„ұлҠҘмқҙ мўӢлӢӨ: мөңлҢҖ л§Ҳм§„ мӣҗлҰ¬ мһҗмІҙк°Җ кіјм Ғн•©мқ„ м–өм ңн•ҳлҠ” л©”м»ӨлӢҲмҰҳмқҙлӢӨ
  • м»Өл„җлЎң л№„м„ нҳ• мІҳлҰ¬: м»Өл„җл§Ң л°”кҫёл©ҙ ліөмһЎн•ң кІ°м • кІҪкі„лҸ„ л§Ңл“Ө мҲҳ мһҲлӢӨ
  • м„ңнҸ¬нҠё лІЎн„°л§Ң м ҖмһҘ: мҳҲмёЎ мӢң м „мІҙ лҚ°мқҙн„°к°Җ м•„лӢҢ м„ңнҸ¬нҠё лІЎн„°л§Ң м°ёмЎ°н•ҳлҜҖлЎң л©”лӘЁлҰ¬ нҡЁмңЁм ҒмқҙлӢӨ

лӢЁм җ

л°ҳл©ҙ л¶„лӘ…н•ң н•ңкі„лҸ„ мһҲлӢӨ:

  • лҢҖк·ңлӘЁ лҚ°мқҙн„°м—җ лҠҗлҰ¬лӢӨ: н•ҷмҠө мӢңк°„ ліөмһЎлҸ„к°Җ O(nВІ~nВі)мқҙм–ҙм„ң, лҚ°мқҙн„°к°Җ 10л§Ң к°ңлҘј л„ҳм–ҙк°Җл©ҙ мӢӨмҡ©м Ғмқҙм§Җ м•ҠлӢӨ. мқҙ мҳҒм—ӯм—җм„ңлҠ” нҠёлҰ¬ кё°л°ҳ лӘЁлҚёмқҙлӮҳ мӢ кІҪл§қмқҙ лӮ«лӢӨ
  • нҷ•лҘ  м¶”м •мқҙ л№„нҡЁмңЁм ҒмқҙлӢӨ: SVMмқҳ м¶ңл ҘмқҖ кІ°м • кІҪкі„к№Ңм§Җмқҳ кұ°лҰ¬м§Җ, нҷ•лҘ мқҙ м•„лӢҲлӢӨ. нҷ•лҘ мқҙ н•„мҡ”н•ҳл©ҙ Platt scaling(probability=True)мқ„ мҚЁм•ј н•ҳлҠ”лҚ°, м¶”к°Җ көҗм°Ё кІҖмҰқмқҙ н•„мҡ”н•ҙ лҠҗл Өм§„лӢӨ
  • мҠӨмјҖмқјл§Ғ н•„мҲҳ: кұ°лҰ¬ кё°л°ҳмқҙлҜҖлЎң Feature Scaling м—ҶмқҙлҠ” м„ұлҠҘмқҙ лӮҳмҳӨм§Җ м•ҠлҠ”лӢӨ
  • н•ҳмқҙнҚјнҢҢлқјлҜён„° нҠңлӢқмқҙ к№ҢлӢӨлЎӯлӢӨ: CмҷҖ gammaлҘј лҸҷмӢңм—җ нғҗмғүн•ҙм•ј н•ҳкі , мөңм Ғк°’мқҙ лҚ°мқҙн„°л§ҲлӢӨ нҒ¬кІҢ лӢӨлҘҙлӢӨ

м–ём ң SVMмқ„ м„ нғқн• к№Ң?

кІ°көӯ м„ нғқ кё°мӨҖмқҖ мқҙл ҮлӢӨ:

  • лҚ°мқҙн„°к°Җ мӨ‘мҶҢ к·ңлӘЁ(мҲҳмІң~мҲҳл§Ң)мқҙкі , кі м°Ёмӣҗмқҙл©°, лӘ…нҷ•н•ң л§Ҳм§„мқҙ мЎҙмһ¬н•  л•Ң вҶ’ SVM
  • лҢҖк·ңлӘЁ лҚ°мқҙн„°мқҙкұ°лӮҳ л№ лҘё н•ҷмҠөмқҙ н•„мҡ”н•  л•Ң вҶ’ лЎңм§ҖмҠӨнӢұ нҡҢк·Җ лҳҗлҠ” нҠёлҰ¬ кё°л°ҳ лӘЁлҚё
  • нҷ•лҘ  м¶”м •мқҙ мӨ‘мҡ”н•  л•Ң вҶ’ лЎңм§ҖмҠӨнӢұ нҡҢк·Җ
  • н•ҙм„қ к°ҖлҠҘм„ұмқҙ мӨ‘мҡ”н•  л•Ң вҶ’ лЎңм§ҖмҠӨнӢұ нҡҢк·Җ лҳҗлҠ” кІ°м • нҠёлҰ¬
рҹ“Ң н•өмӢ¬ мҡ”м•Ҫ

  • SVM: л‘җ нҒҙлһҳмҠӨ мӮ¬мқҙмқҳ л§Ҳм§„мқ„ мөңлҢҖнҷ”н•ҳлҠ” л¶„лҘҳкё°
  • м„ңнҸ¬нҠё лІЎн„°: л§Ҳм§„ кІҪкі„м—җ кұём№ң лҚ°мқҙн„° нҸ¬мқёнҠё. мқҙкІғл§ҢмңјлЎң кІ°м • кІҪкі„к°Җ кІ°м •лҗЁ
  • C нҢҢлқјлҜён„°: мҳӨл¶„лҘҳ лІҢм җ к°•лҸ„. нҒҙмҲҳлЎқ л§Ҳм§„ мўҒмқҢ(кіјм Ғн•©), мһ‘мқ„мҲҳлЎқ л§Ҳм§„ л„“мқҢ(кіјмҶҢм Ғн•©). к·ңм ң к°•лҸ„(О»)мқҳ м—ӯмҲҳ
  • м»Өл„җ нҠёлҰӯ: кі м°Ёмӣҗ л§Өн•‘ м—Ҷмқҙ лӮҙм Ғл§ҢмңјлЎң л№„м„ нҳ• л¶„лҘҳ. RBFк°Җ кё°ліё
  • gamma: RBFмқҳ мҳҒн–Ҙ лІ”мң„. нҒҙмҲҳлЎқ кІҪкі„ ліөмһЎ(кіјм Ғн•©)

л§Ҳм№ҳл©°

нҷ•лҘ (лЎңм§ҖмҠӨнӢұ нҡҢк·Җ), кұ°лҰ¬(KNN), л§Ҳм§„(SVM) вҖ” м„ё к°Җм§Җ м „нҳҖ лӢӨлҘё м ‘к·јмңјлЎң л¶„лҘҳн•ҳлҠ” лІ•мқ„ л°°мӣ лӢӨ. н•ҳм§Җл§Ң мқҙ лӘЁл“  лӘЁлҚём—җлҠ” кіөнҶөм Ғмқё к·јліё л¬ём ңк°Җ мһҲлӢӨ. лӘЁлҚёмқҙ лӢЁмҲңн•ҳл©ҙ лҚ°мқҙн„°мқҳ нҢЁн„ҙмқ„ лӘ» мһЎкі , ліөмһЎн•ҳл©ҙ л…ёмқҙмҰҲк№Ңм§Җ мҷёмҡҙлӢӨ. лӢӨмқҢ кёҖм—җм„ңлҠ” нҺён–Ҙ-л¶„мӮ° нҠёл Ҳмқҙл“ңмҳӨн”„лҘј нҶөн•ҙ мҷң лӢЁмқј лӘЁлҚёмқҖ н•ңкі„к°Җ мһҲлҠ”м§Җ, к·ёлҰ¬кі  мқҙ н•ңкі„лҘј м–ҙл–»кІҢ м§„лӢЁн•ҳлҠ”м§ҖлҘј м•Ңм•„ліёлӢӨ.
